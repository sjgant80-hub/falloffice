## How it works

```
1. Open index.html in any browser
2. Click an agent card → workspace opens
3. Type your request in the chat
4. AI cascade processes it:
   Gemini (primary) → DeepSeek (fallback) → Echo (offline)
5. Response appears with agent styling
6. Relevant metric increments
7. Chat history saves to localStorage
8. Switch between agents — each has independent history
```

### AI cascade

The tool always works, even with no internet:

- **Primary: Gemini 2.0 Flash** — fast, capable, free tier available
- **Secondary: DeepSeek** — strong alternative, very cheap
- **Fallback: Echo** — offline mode that outlines what the agent would do and guides you through structured thinking

Set API keys in Settings. No keys? It still works in offline mode.

## Quick start

1. Open `index.html` in Chrome, Edge, Safari, or Firefox
2. Click **⊙ Settings**
3. Add a Gemini API key (free at [ai.google.dev](https://ai.google.dev))
4. Enter your company name, description, and ICP
5. Close settings
6. Click **Scout** → type a company name → get structured research
7. Click **Composer** → paste the research → get personalised outreach

That's the workflow. Scout finds them. Composer writes to them. Sequencer plans the follow-up. Analyst tells you which deals to focus on. Writer creates content. Social distributes it. SEO plans the calendar. Ads drives paid traffic.
